# Anthony Luan
Este projeto marca o meu primeiro passo no desenvolvimento de soluções de automação em Python. Ele foi criado com o objetivo de otimizar tarefas repetitivas no ambiente corporativo, tornando processos mais rápidos, seguros e eficientes.

A ferramenta realiza de forma automática o download de um arquivo com dados da empresa e, em seguida, processa essas informações para enviar relatórios e mensagens por e-mail sem necessidade de intervenção manual. Todo o fluxo — da coleta dos dados ao envio das mensagens — é executado de maneira integrada, garantindo agilidade e precisão nas operações.

Durante o desenvolvimento, pude aplicar conceitos fundamentais de programação, manipulação de arquivos, uso de bibliotecas externas, automação de rotinas e envio de e-mails via script. O projeto não apenas demonstra minha evolução técnica, como também reflete minha capacidade de identificar problemas reais e construir soluções práticas para resolvê-los.

Este é apenas o começo da minha trajetória, e sigo motivado a aprimorar e desenvolver novas automações cada vez mais completas e inteligentes.
